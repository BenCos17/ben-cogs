A cog for fetching images from the Martine API, including memes, wallpapers, subreddit images, ship images, and osu! profile cards.

# [p]martinememe
Get a random meme from Martine API.<br/>
 - Usage: `[p]martinememe`

# [p]martinewallpaper
Get a random wallpaper from Martine API.<br/>
 - Usage: `[p]martinewallpaper`

# [p]martinesubreddit
Get a random image from a specified subreddit using Martine API.<br/>
 - Usage: `[p]martinesubreddit <subreddit>`

# [p]martineship
Generate a ship image with two Discord users using Martine API.<br/>
 - Usage: `[p]martineship <user1> <user2>`

# [p]martineosuprofile
Generate an osu! profile card using Martine API.<br/>
 - Usage: `[p]martineosuprofile <username>`
